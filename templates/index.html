Used to find the combinations of adduct base on the neutralmass mass and mass from the unifi devices.
How to use:
1. Fill all the blanks with parameters:
File:
- negative_unifi.csv for the elements database of negative cases.
- positive_unifi.csv for the elements database of positive cases.
NeutralMass:
- The paramenter should be float number xxx.xxxxx.
Unifi_number:
- Number from the unifi machine.
Hydro(s):
- Number of Hydros can be in the combinations.
- The case that the addduct does not contain Hydro is always calculated.
Element(s):
- The number of times the elements can be duplicated in the adduct combi.
Mass Error:
- The acceptable error which can be approved, the smaller it get the more accurate the results.
- Normal paramter is: 0.00001
Hydro Exact:
- should equal 1.007825 as standard value.
2. Click Convert.
3. The result will be shown as Json format.
